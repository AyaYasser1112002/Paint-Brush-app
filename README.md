Overview
The Paint Brush Applet is a simple Java-based drawing application built using Applet and AWT. It allows users to draw basic shapes, sketch freely, erase, undo, and change colors dynamically. This project is inspired by classic paint programs, providing an interactive graphical experience.

✨ Features
✅ Draw Basic Shapes – Line, Rectangle, and Oval
✅ Free-Hand Drawing – Sketch using mouse dragging
✅ Color Selection – Choose between Red, Green, and Blue
✅ Eraser Tool – Remove parts of the drawing
✅ Dotted & Filled Shapes – Switch between solid and outlined shapes
✅ Clear Canvas – Reset the drawing area

🛠️ Technologies Used
Java Applet & AWT – Graphical Interface
Event Handling – Mouse & Keyboard Interactions
Data Structures – ArrayList for managing shapes
Git & GitHub – Version Control

🚀 Setup & Usage
1️⃣ Clone the Repository
git clone https://github.com/AyaYasser1112002/Paint-Brush-App.git
cd Paint-Brush-App

2️⃣ Compile the Java Files
javac PaintBrushApplet.java

3️⃣ Run the Applet
appletviewer PaintBrushApplet.java
